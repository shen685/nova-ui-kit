# Nova UI Kit ✨

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![React](https://img.shields.io/badge/React-18.x-blue.svg)](https://reactjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.x-38B2AC.svg)](https://tailwindcss.com/)

**Nova UI Kit** is a minimalist, high-performance UI component library designed for modern web applications. Built with **React**, **TypeScript**, and **Tailwind CSS**, it focuses on accessibility, performance, and developer experience.

## 🚀 Features

- 🎨 **Modern Design**: Clean and minimalist aesthetic.
- ♿ **Accessible**: Built with accessibility (A11y) in mind.
- ⚡ **High Performance**: Optimized for speed and small bundle size.
- 🛠️ **Customizable**: Easily themeable with Tailwind CSS.
- 📦 **Tree-shakable**: Only import what you need.

## 📦 Installation

```bash
npm install @nova-ui/kit
# or
yarn add @nova-ui/kit
```

## 🛠️ Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
